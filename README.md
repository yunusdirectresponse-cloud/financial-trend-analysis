# Financial Trend Analysis (2022–2024)

This project analyzes financial data from Microsoft, Apple, and Tesla using Python.

## Objective
To identify revenue and profitability trends and generate insights for decision-making.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Microsoft shows strong and accelerating growth, especially in 2024.
- Apple remains stable with slight decline in profitability.
- Tesla experienced rapid growth followed by stabilization.
- Companies are at different growth stages, useful for strategic decisions.

## Output
Includes:
- Year-over-year growth calculations
- Data table
- Revenue trend visualization

## Conclusion
This analysis demonstrates how financial data can be transformed into actionable insights using Python.
